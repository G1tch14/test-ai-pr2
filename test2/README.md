# Hello World Console Application

A simple console application that prints "Hello, World!" in multiple languages.

## How to Run

### Java Version
1. Make sure you have Java installed (JDK 8 or higher)
2. Compile the Java program:
   ```bash
   # On Windows:
   build.bat
   
   # On Unix/Linux/Mac:
   chmod +x build.sh
   ./build.sh
   
   # Or manually:
   javac HelloWorld.java
   ```
3. Run the compiled program:
   ```bash
   # On Windows:
   run.bat
   
   # On Unix/Linux/Mac:
   chmod +x run.sh
   ./run.sh
   
   # Or manually:
   java HelloWorld
   ```

### Python Version
```bash
python hello.py
```

Or make it executable and run directly:
```bash
chmod +x hello.py
./hello.py
```

## Requirements
- **Java Version**: Java JDK 8 or higher
- **Python Version**: Python 3.x

## What it does
The application simply prints "Hello, World!" to the console when executed.
