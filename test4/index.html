Hello World Application

A simple console application built with Java

About This Project

This is a basic console application that demonstrates a simple "Hello World" implementation in Java.

Project Features:

  • Simple console output
  • Built with Maven
  • JUnit 5 for testing
  • Clean project structure

How to Run

To build and run the application:

# Build the project
mvn clean compile

# Run the application
mvn exec:java -Dexec.mainClass="com.example.helloworld.HelloWorldApp"

# Or use the provided scripts:
./build.sh
./run.sh

Project Structure

The project follows standard Maven directory structure:

src/main/java/com/example/helloworld/HelloWorldApp.java
src/test/java/com/example/helloworld/HelloWorldAppTest.java
